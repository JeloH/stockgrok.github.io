Stockgrok

A sonic chart analysis tool for Google Chrome

Try out the Stockgrok prototype on Google Chrome

Stockgrok is a web-based tool that provides auditory counterparts to visual cues in charts that are used to assess financial securities. Compare the price of any security of interest to its 50 Day Simple Moving Average purely through sound. Stockgrok uses a unique set of audio outputs that enable hearing the distance between lines, intersection points, and price position above or below its moving average. Stockgrok is an inclusive solution designed to empower people with visual impairments assess financial trends and make buy or sell judgements non-visually.

Note: While we welcome your feedback, please note that this project is not currently being actively maintained.
